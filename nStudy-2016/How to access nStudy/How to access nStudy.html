You can access nStudy by following these steps:
  1. Download latest version of Chrome web browser

  2. Open the Chrome web browser.

  3. In the address field, enter this URL: https://chrome.google.com/webstore/detail/nstudy/mmmiodfchlgjfnbnpgaecgnmhbdpceki

  4. A message will appear asking you to install the nStudy add-on. Be sure you follow the steps to install.



  5. Click nStudy icon on the top right of the Chrome's address bar. It is all black when inactive


  6. To create an nStudy account, go to: https://nstudy.educ.sfu.ca/ns4/user/login/
  7. Click "Create New Account"
