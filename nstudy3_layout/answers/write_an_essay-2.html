You can write an essay by following these steps:

  1. Mod-click anywhere on your Map



  2. Choose “Document” from the context menu

  3. Type over "untitled document" to give your essay a title

  4. Then you double click your “document” to open it in the main browser.


Tip: An nStudy document has common features for editing text. Some of these features are:
  1. Insert a table
  2. Insert an Image
