nStudy is an online learning tool. It helps you organize information you study and manage that information. It also keeps track of your work. nStudy requires you to install an extension to your Chrome web browser. See the section "How do I access nStudy?". In this description, we italicize terms of the vocabulary of nStudy and how it works.

nStudy has two different views: a "Reading view" and a "Maps view"

  1. In the Reading view, you can create objects such as Highlights and Notes with different templates.

  2. Each object you create, stays attached to the location in which it was created
